Linking It up: the Easy Version

There are two potential pieces of HTML that you could be using in your navigation schemes. The one you would like to use looks like this:

<a href="/jump_to_id/d762c2a2f8b94b40836ebc75d1c5f67e">Linked Thing</a>

...where "Linked Thing" is going to be text, or an image, or part of an image map. This is called a "jump to ID" link. The edX documentation on links has all the info about them, and you've probably seen the description in Studio under "Unit Location" more than once. These links are great in that they work for later runs of the same course. Good stuff, A++, would use again, try this first.
