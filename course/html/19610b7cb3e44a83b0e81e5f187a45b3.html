Flowchart of happiness

As mentioned previously, the setup for cohorts is pretty logical but it does require each step to be approached in sequential order. If you try and do things in a different order, the platform will simply say no! EdX has limited error messages, and you might not be able to figure out exactly where you went wrong.

Whilst quite a lot of this could be tackled through trial and error, we have put together a neat flow chart to guide you through the process, with all its nuances and intricacies. That being said, whilst there are a lot of steps, each step is pretty gosh-darn quick to complete, so the whole process can be finished in a matter of minutes (providing you know which configurations you’re after).

In reality, the thing that takes the most time is chasing people to sign up to the platform and course, when using manual allocation - automatic allocations require minimal setup time.

We will go into more detail about the why and when of ‘setting up content experiments’ in another section. This section is designed to help you better understand all the configuration options available should you wish or need to use them.

The cohort setup flowchart, with a lot more detail at each step, is available to view via the link on image or download.

Tip: Studio Warnings

Have a play around in studio and try to do things in a different order. You’ll see the platform has some useful warning buttons to help you along the way.
