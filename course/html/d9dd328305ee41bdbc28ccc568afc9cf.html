Now you’ve worked out what you want to do with cohorts and you’ve considered the platform variables that can be changed, it’s time to go on platform.

The following video shows a really simple example of setting up cohort for a common usage - creating smaller discussion groups - and covers the basic workflow for setting up automatic allocation cohorts. The ‘Cohort Strategy’ for the demo is as follows:

Demo course strategy

    • Rationale: to improve user engagement with discussion boards by making them less overwhelming in size
    • 3 cohorts required: Cohort A, Cohort B & Cohort C 
    • Automatic allocation - it doesn’t matter who is in each group, size is most important
