What goes into the CSS?

In order to display your style sheet, you will need to create a .css file. This is a plain text file with a .css extension. We will cover the setup in a bit more detail on the next page but first, let's have a look behind the scenes at the code that makes the style happen!

Below are a few examples from the previous demo, showing the code used in the .css file and html in edX Studio editor:

  • Indented bullets - Simple CSS
  • Header underlined - HarvardX
  • Purple text box 1 - HarvardX
  • Purple text box 2 - HarvardX Alternate
  • Image with a border and caption - Simple CSS
