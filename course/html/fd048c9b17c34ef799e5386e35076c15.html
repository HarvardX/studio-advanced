HX-JS

HX-JS is a Javascript library that collects a lot of the tricks and tools that HarvardX has written for our courses in one place, to make them easier to implement and troubleshoot. You've seen it at work all over this course:

  • Clickable image maps
  • The image carousel
  • Pop-up links in videos
  • Show/hide toggles like in the choose-your-own-adventure problem
  • Little cosmetic tweaks like the header underline, colored boxes, and floating images.
  • Also there's a demonstration of all of these on the next page if you like.

The current version of the code for this can be found on the hx-js GitHub repository, along with detailed instructions for implementation. It's covered under the MIT License so you can do whatever you like with it.

Installation

  1. Download the hx-standard.zip archive. Unzip it and put every file from it into your Files & Uploads section. This is the standard minimum set of tools.
  2. Download the HXJS Helper.zip archive. Unzip it and put every file from it into your Files & Uploads section. This will let you use video links, pop-up problems, Slick-style image sliders, and intro.js text-based walkthroughs.

Once you've done that, copy the lines below into a Raw HTML element:

<script src="/static/hx.js"></script>
<link rel="stylesheet" type="text/css" href="/static/hx.css">

That will enable hx.js for all components on that page. It should be doing exactly nothing so far. You can test whether it's working by entering the Konami Code. If you get 30 lives, it's working. You should put it just once on every page where you want it to work.

HX-JS mostly works by you setting specific classes on things, and it finding those classes and attaching listeners to the thing that has the class. It tries not to load things unless they might be needed, so if there's no video on the page, it won't load the video-related files.

Options

There are a bunch of options you can set for HX-JS, all of which are described on the GitHub repo. You can either set them in the HXGlobalOptions.js file, or you can set them locally on your page, like this:

<script>
var hxLocalOptions =  {
  // Marks all external links with an icon.
  markExternalLinks: false,

  // Options for the Slick image slider
  slickOptions: {
    arrows: true,
    dots: false,
    slidesToShow: 1,
    slidesToScroll: 1
  },
};
</script>

Again, all the options can be found in the GitHub repo. Scroll down to the bottom there for the full list.
