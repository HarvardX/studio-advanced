Getting your new library into a course

So your content library is fully populated with required questions and ready to be included within your course. Brilliant! Below is a whistle-stop guide to getting your randomised questions displayed in-course:

Step 1: Enable "library_content" in Advanced settings of your course

Step 2: Add a randomised content block as an advanced component on an assessment unit/page

Step 3: Select a library and state the number of questions required

There are two important bits of information in the component block settings: library to reference and number of questions to display. 'Count' refers to the number of items (most typically questions) you would like to display to learners. Under 'library' you can select the required content library for your section - it will list every library you have access to, so this is where the unique references come in handy!

Once this is saved, your randomised component will be enabled and learners will see a subset only of the full list of questions in the library. You can double check the questions in the library by pressing 'view' on the block.
