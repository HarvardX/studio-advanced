No Live Demo, But...

Because this is in Python, we can't easily give you a live demo online. You have to download the scripts and run them yourself. That course is stored on a public GitHub repo, and it has a little of everything so that you can test it out. If GitHub isn't up, you can download the tar.gzip file here, though we had to take out one or two things to get it to fit in Files & Uploads.
