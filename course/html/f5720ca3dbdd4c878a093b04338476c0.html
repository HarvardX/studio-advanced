The javascript will turn this set of divs into a list of static links, for folks who can't see or get to the pop-up links.

You can set the following options. Any that you don't set will default to the values shown below. Check out the official jQuery UI showcase for more detail on the effects and directions below.

var hxLocalOptions =  {
  VidLinkOptions: {
    hideLinkAfter: 5,           //seconds
    effect: 'slide',            // As per standard jQuery options
    hide: {'direction':'down'}, // As per standard jQuery options
    show: {'direction':'down'}, // As per standard jQuery options
    speed: 500,                 // Miliseconds
    location: 'bl'              // First letter is b/t (Bottom/Top), 
                                // second letter is l/r (Left/Right).
  }
}
