Warning: Overrides

The Advanced Settings page says "Any policies you modify here override all other information you've defined elsewhere in Studio." This is not always true. In many cases, such as the items under "Assignment Settings (Individual)", the local settings are the ones that take precedence.

For example, if your course is set to Show Answer: Finished, and an individual problem is set to Show Answer: Always, that problem's local rule wins. Even if you change the course-wide setting, that problem will still be set to Show Answer: Always.

Be sure to test your settings changes to make sure they have the effects that you expect.
