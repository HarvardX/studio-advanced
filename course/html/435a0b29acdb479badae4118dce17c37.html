Image Indicator

As we showed earlier in this section, you can use a walkthrough when your learners first arrive in the course to give them an overview of the edX platform. We also showed you a walkthrough that points out useful features in code. Here's another demonstration: using a walkthrough to indicate important parts in an image. Ready to see it in action?


Here's the code for this page as a zipped HTML file. A few important things to note:

  • Because we created all the elements that the walkthrough touches, we don't need to add the data elements via javascript - we can just write them into the HTML from the start.
  • We're positioning invisible <div> tags over the image in order to target different parts of it. We need to position them by percentage so that smaller screen sizes still have the boxes in the right place. You cannot run a walkthrough with an image map, because the <area> tags aren't actually positioned over the image.
  • There's some extra css on this page to handle the opacity of the indicator, and the opacity of the dark background.
