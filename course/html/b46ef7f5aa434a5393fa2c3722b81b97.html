Simple Question Markup

Here is a cribsheet for reference when improving your basic question types in edX. Remember the spaces before and after text in feedback brackets.

Item Markup required
Adding an extra instruction line to a question >> question || instruction <<
(after the question text)
Adding a 'hint' button || text || (at the bottom of the MCQ section)
Feedback based on a specific submitted answer {{ text }} (after the answer)
Feedback based on selected or unselected checkboxes (multiple answers) {{ selected: text }{ unselected: text }} (after the answer)
Randomising questions on a page See section 4.4 Content Libraries
Randomising answer order display for a given question (!) Exclamation mark in your first answer's parens.
See also Settings → Randomisation
Fixing an answer in place while others are randomised (@) At-sign in that answer's parens.
Often used to pin "all of the above" at the bottom.
Deciding whether to show the answer upon submission Settings → Show Answer
Tolerances on numerical submission = number +- tolerance (after the answer)
