Imagine that you have a course with dozens of assignments, made by different people on different days. They might have the right number of attempts, the right "show answer" settings... or they might not. Did you remember to set all of your videos to allow transcript downloads, or just some of them? Changing the Advanced Settings only takes effect when those settings aren't overridden locally. Do you need to go through everything one component at a time?

Luckily there's a way to make these kinds of changes - it just requires some scripting. Behind the scenes, edX uses the OLX format to describe its courses. OLX is a type of XML, which makes it possible to write programs that parse and alter edX courses. We'll introduce a small collection of these, which you can run on your own computer in order to make course-wide changes.

The goal: Change settings for every problem or video in your course quickly and programmatically.
Prerequisites: You should be comfortable running command-line instructions. These scripts require Python 3, and make use of a handful of commonly available python packages. If none of that makes sense to you, this is probably not the tool for you to use.
Project length: 30 minutes to 2 hours for initial setup and installs. 5 minutes or less to run a script.
Ingredients:

  • Your course "tarball", as downloaded from the Export page in Studio.
  • At least one script from the HX-XML repository, such as SetMaxAttempts.py. You can get these by downloading individual scripts, or by cloning or downloading the whole repository.
  • To reupload the course, you will want a utility that can create .tar.gz files.
    • On Mac and Linux machines, this can be done via the command line.
    • On Windows, you may want to consider using 7-Zip.

Original Source: HX-XML repository. Available for use without restriction, but not intended for redistribution.
