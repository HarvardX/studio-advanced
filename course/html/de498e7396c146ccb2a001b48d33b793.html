Setup

Before you run batch-processing scripts, you'll want to take the following steps:

1. Ensure you're running Python3

Our scripts were written for version 3+ of Python. Version 3.7.x is the current state of the art, but version 3.5 or later should work just as well. Python has a Beginner's Guide that will walk you through the process, or you can download the current version if you're comfortable doing so.

Warning: If the installer gives you the option to replace your current version of Python, do not replace your existing version of Python. Doing so can make your computer unusable. Install the new one but keep the old one.

2. Install packages

Some of our scripts rely on external packages:

  • Beautiful Soup version 4, an improved html parser. You can install it via pip, or find it in the bs4 folder from the HX-XML repository.
  • lxml, an XML parser. You can install this via pip.

3. Get the scripts

The easiest way to do this is to clone the HX-XML repository. If you don't yet use GitHub, you can register for free and use the desktop app, but you can also download the whole repository as a zip file or obtain the scripts one-by-one by downloading them from the repository. You don't need to register for that.
