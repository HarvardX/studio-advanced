Applications: Other Tricks

This tool is really one of a whole class of "dumb internet hack" tricks that you can pull using various free providers and a little bit of parsing and/or display code. Here are a few related hacks that you might find useful.

  • Distributed Experiments: Embed a form in a science or statistics course, asking every learner to roll a die, or flip a coin, or do a quick experiment. Gather the data, build a graph, and publish that graph so you can embed it in the course. Learners can see the values changing every time they come back to the course, as more people enter their results.
  • Color a Map: The table trick is pretty easy in comparison to this one... so it's a good thing we built this one first! The datamap folder in HX-JS will let you color a blank SVG world map based on country-by-country data in a CSV file. You can load specific data files, or you can have people fill out a form - though you might have to make a few extra sheets or write some extra code to do some processing for you. Special thanks to whomever made the SVG world map on Wikimedia Commons.
  • Filterable Table: There's no reason you need to use a form in this setup. If you just want to show tabular data and let your learners filter it, you can save that data in SVG format and just use the bottom code block on the previous page.
