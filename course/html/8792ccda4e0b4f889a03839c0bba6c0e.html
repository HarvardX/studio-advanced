Examples: Sketches of landing page layout ideas

The examples below offer a few different ideas, plus commentary, of how landing page layouts could (and probably should not) be designed. You can click each one for a larger image.

This is the typical layout using a text announcement from the course team.

You could think about using a single main image like an infographic or images that link to course content as part of an alternative navigation strategy.

If you wanted to draw attention to a really important resource, you could feature it in a newsletter style update.

It is possible to embed video in the update, but remember to use an iframe embed method and be mindful of the width limitations.

This is an example of content saturation! The update is so long that the course outline will be obscured from view and could be quite confusing for the learner.
