Selecting Advanced Problem Types


Easy

Day

EdX offers a wide range of assessments. Five of them (Multiple-choice, Checkbox, Dropdown, Numerical, and Text) are "Common" problem types, and can be written in a simple markdown format. These will cover a wide variety of needs, and will be the bread and butter of assessment in most courses. However, sometimes you want to ask questions that don't easily map to one of these problems. That's when you turn to Advanced Problem Types.

This section presents a few types of Advanced Problem, so that you can better understand what's available and select what to use throughout your course.

Prerequisites: To learn what's available, nothing. To implement them, you need familiarity with HTML, and sometimes with Python and Javascript. You will also need to be a content expert in this course, or work with someone who is. Someone could implement advanced questions without needing to know the course material well, but you can't select which ones to implement unless you understand the course subject.
Project length: 60 minutes to learn, 60 minutes to decide and create a few mockups, 30+ minutes to build each advanced problem.
Ingredients: Many advanced problems are already supported within edX. However:

  • Some advanced tools and problems will need to be activated via Advanced Settings.
  • For some Javascript Problems, you will want to obtain code from Stanford's JSInput Repository. You will often need to edit JSON or XML files to use these.
