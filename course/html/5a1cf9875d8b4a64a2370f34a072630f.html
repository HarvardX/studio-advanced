So, you've decided you want to get more familiar with the code behind your course. Whilst the edX Rich Text editor is available on every page, it can often hide some uncomfortable truths about the code underneath. The HTML editor gives you far more functionality and visibility of what's actually on the page.

But why should you bother using raw HTML when the Rich Text editor is soooooo easy? Hopefully, this section will give you some reasons why and gently persuade you over to the better side!
